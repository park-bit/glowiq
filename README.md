## Configuration üîß

### AI Integration
1. **Get an API Key**:
   - Sign up for [Hugging Face](https://huggingface.co/) or [OpenAI](https://openai.com/).
   - Generate an API key from your account settings.

2. **Add API Key**:
   - Open `lib/services/ai_service.dart`.
   - Replace `YOUR_API_KEY` with your actual key.
